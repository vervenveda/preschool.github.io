## Overview

**Sunny ABCs & 123s — Storytime** is a colorful, self-contained preschool learning application built as a single HTML file.

Young learners explore the alphabet and early numbers through short rhymes, cheerful visual symbols, spoken narration, physical movement prompts, page-turn animations, and simple progress rewards.

The current version is designed to be easy to understand, welcoming to young learners, and usable without accounts, advertising, analytics, tracking, or external game services.

Progress and preferences remain inside the learner’s browser.

---

## Current Learning Content

### Alphabet Storybook

The alphabet section includes all **26 letters from A through Z**.

Each page contains:

- A large featured letter
- A familiar vocabulary word
- A matching visual symbol
- A short four-line rhyme
- A movement or imagination prompt
- A spoken-word button
- Tap-to-read rhyme lines

Examples include:

- A is for Apple
- B is for Balloon
- G is for Garden
- M is for Moon
- R is for Rainbow
- Z is for Zoo

### Number Storybook

The current number section includes **numbers 1 through 10**.

Each page contains:

- A large featured numeral
- The written number word
- A counting-themed rhyme
- A movement or counting prompt
- Spoken narration
- Page navigation
- Progress tracking

## Privacy and Technical Design

Sunny ABCs & 123s is designed as a sovereign, privacy-respecting learning tool.

- No accounts
- No advertising
- No analytics
- No tracking
- No required registration
- No external JavaScript libraries
- No student information sent to outside services
- No required internet connection after download
- Progress remains on the learner’s device
- Preferences are stored locally in the browser

## How to Use

1. Open `Sunny_ABCs_123s_Storytime_Upgraded.html` in a modern browser.
2. Select **Alphabet** or **Numbers**.
3. Use the page arrows, navigation dots, keyboard arrow keys, or swipe gestures to move through the book.
4. Select **Say Word** to hear the featured word.
5. Select **Movement Prompt** to hear the physical activity.
6. Tap a rhyme line to hear it read aloud.
7. Turn on **Song Mode** to hear the complete rhyme one line at a time.
8. Use the **Theme** control to change the visual atmosphere.

# Recommended Upgrade Roadmap

The older version already has a strong storybook foundation. The following recommendations would make it crisper, softer, more dynamic, and more instructionally complete while preserving its gentle preschool character.

---

## 1. Softer Typography and Crisper Layout

Recommended visual improvements:

- Use a softer rounded font stack
- Improve spacing between the picture, symbol, word, prompt, and rhyme
- Reduce visual crowding around navigation dots
- Increase text hierarchy and contrast
- Enlarge controls for small hands
- Refine mobile spacing
- Make the storybook feel more open and breathable
- Use gentler shadows and cleaner card edges
